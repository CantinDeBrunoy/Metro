### Lancement du projet
***
Le projet peut être lancé grâce au fichier Makefile avec les commandes suivantes 
/!\ Attention : les commandes a suivre sont a entrer dans un terminal sous environnement Linux /!\

> make (compile le projet et crée les fichiers .build)
> make run (lance le projet)

//après avoir lancé le projet avec "make run"
> a (affiche/cache l'acpm sur le graphe)

> make clear (supprime les fichiers .build)
